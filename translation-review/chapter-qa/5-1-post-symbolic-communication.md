# 5-1-post-symbolic-communication QA

## 요약 판정

큰 구조 누락이나 장 단위 의미 붕괴는 보이지 않는다. 제목, 소제목, 수평선, 목록, 링크 URL, 각주 참조/정의의 순서는 원문과 대응한다. 스터디용으로는 충분히 읽히지만, 일부 전문 용어가 직역투이거나 한국어 학술 독자에게 더 자연스러운 통용어가 있어 P2/P3 중심의 윤문을 권장한다.

## P0 구조/누락

없음. 원문 기준 H1 1개, H3 4개, 수평선 1개, 목록 16개, 각주 정의 13개가 번역문에도 대응한다. 원문에 이미지 경로, figure 구조, code fence가 없으며 번역문에도 새로 생기거나 누락된 항목은 없다. 링크 URL도 모두 보존되어 있다.

## P1 의미 오역

없음. 핵심 논지, 사례 순서, 위험/한계 논의는 대체로 원문 의미를 따라간다.

## P2 용어/문체

- [P2] 번역 9 / "Not coincidentally" / 현재 "우연은 아니게도"는 한국어 관용으로 어색함 / "우연이 아니게도" 또는 "공교롭지 않게도"로 수정
- [P2] 번역 29 / "Psychedelics" / 현재 "환각제"는 다소 협소하고 낙인적 뉘앙스가 강할 수 있음 / 장 전체 용어를 "사이키델릭 물질" 또는 "환각성 물질"로 통일 검토
- [P2] 번역 38 / "Cambrian explosion" / 현재 "캄브리아기적 폭발"은 통용 표현에서 벗어난 직역감이 큼 / "캄브리아기 대폭발" 권장
- [P2] 번역 38 / "combine in supermodular ways" / 현재 "초모듈적인 방식으로 결합"은 독자에게 의미가 불투명함 / "상호보완적으로 결합해 시너지를 내는 방식" 또는 "초모듈적, 즉 상호보완적 방식"으로 풀기
- [P2] 번역 46 / "moral bio-enhancement (MBE)" / 현재 "도덕적 생체 향상"은 용어성이 약하고 의미가 덜 선명함 / "도덕적 생명공학적 향상(MBE)" 또는 "도덕적 생물학적 향상(MBE)" 검토
- [P2] 번역 58 / "experiential osmosis" / 현재 "경험적 삼투"는 직역으로 뜻이 닫힘 / "경험의 삼투"로 두되 뒤에 "직접 경험의 흡수"를 덧붙이는 방식 검토
- [P2] 번역 66 / "alternate reality, or simulation" / 현재 "대안적 현실"은 긍정적 선택지처럼 읽힐 수 있음 / 문맥상 "대체 현실 또는 시뮬레이션" 권장

## P3 취향/윤문

- [P3] 번역 5 / "a poignant example" / 현재 "가슴에 와닿는 사례"는 감상적이고 구어적임 / "인상적인 사례" 또는 "강렬한 사례"로 정돈
- [P3] 번역 17 / "unborn child" / 현재 "태어나지 않은 아이"는 풀어 쓴 표현이라 문장이 길어짐 / "태아"로 간결화
- [P3] 번역 23 / "chanting" / 현재 "찬트"는 외래어 느낌이 강함 / 맥락에 따라 "염송", "구송", "성가" 중 하나로 정리
- [P3] 번역 41 / "sense of agency over a tail" / 현재 "꼬리에 대한 행위주체감"은 정확하지만 딱딱함 / "꼬리를 움직이는 주체감"처럼 독해 친화적으로 풀기
- [P3] 번역 58 / "misunderstandings are a choice and empathy abound" / 현재 "오해가 선택 사항이 되고 공감이 풍부한"은 의미는 전달되나 기계적임 / "오해는 피할 수 없는 조건이 아니라 선택에 가까워지고, 공감은 넘쳐나는"처럼 문장화

## 보류/주의

- [보류] 원문과 번역문 모두 `[^Lowry]` 각주 정의에 콜론이 없다. 원문 보존으로 보이며 번역 과정의 누락은 아니지만, 표준 Markdown 렌더러에서는 각주 정의로 인식되지 않을 수 있다.
- [보류] `The Park of Aging`, `Adumu`, `DiverCity`, 도서/논문명 등 고유명사와 서지 정보는 대체로 원문 형식을 유지한다. 프로젝트 용어집이 별도 한국어 병기를 요구한다면 첫 출현부에만 병기하는 방식이 좋다.
- [보류] 각주 서지 중 다수는 영어 원문으로 유지되어 있다. 장 전체 정책상 서지 번역을 최소화하는 방향이라면 현 상태 유지가 적절하다.
