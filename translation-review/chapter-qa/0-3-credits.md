# 0-3-credits QA

## 요약 판정

조건부 통과. 원문과 번역 모두 단일 `#` heading과 한 개 본문 단락으로 구성되어 있고, 큰 누락이나 순서 변경은 없다. 링크 URL, 이미지 경로, code fence, figure 구조는 원문과 번역 모두 없음. 다만 제외 대상 설명에서 두 주 저자와 추천/지지 기여자의 제외 사유가 원문보다 모호해졌고, `community currency`, `quantitative holdings` 등은 스터디용 번역으로 조금 더 자연스럽게 다듬을 여지가 있다.

## P0 구조/누락

- 없음.

## P1 의미 오역

- [P1] 본문 1문단 후반 / "excludes the two primary authors as they are sufficiently highlighted above, as well as anyone who only received credit for endorsing, as they are listed..." / 현재 번역은 "이들은 이 책의 첫머리나 표지에..."가 두 주 저자와 추천사 기여자 모두를 가리키는 듯해, 두 주 저자는 위에서 충분히 강조되었고 추천/지지 기여자는 책 앞부분 또는 외부에 등재되어 있다는 원문의 별도 사유가 흐려짐 / 두 집단의 제외 사유를 각각 분리해 번역하는 방향 권장.

## P2 용어/문체

- [P2] 본문 1문단 첫 문장 / "a community currency" / 현재 번역 "커뮤니티 통화"는 의미는 전달되지만 한국어에서 전화 통화와의 중의성도 있고 직역감이 큼 / "커뮤니티 화폐" 또는 "공동체 화폐"처럼 화폐성 용어로 조정 권장.
- [P2] 본문 1문단 중반 / "quantitative holdings of ⿻" / 현재 번역 "⿻의 정량적 보유량"은 의미는 맞지만 학술 번역투가 강하고 독자에게 딱딱하게 읽힘 / "보유한 ⿻의 양" 또는 "⿻ 보유량"처럼 간결하게 조정 권장.
- [P2] 본문 1문단 후반 / "only received credit for endorsing" / 현재 번역 "추천사에 대해서만 크레딧을 받은 사람들"은 `endorsing`을 추천사 작성으로 좁혀 읽히게 할 수 있음 / 책 표지·첫머리의 endorsement 맥락을 살려 "추천/지지에 대해서만 크레딧을 받은 사람들" 등으로 완화 권장.

## P3 취향/윤문

- [P3] 본문 1문단 첫머리 / "The below represents" / 현재 번역 "아래는 ... 나타내며"는 이해 가능하지만 주어가 다소 비어 보임 / "아래 내용은 ... 보여준다"처럼 명사 주어를 보강하면 자연스러움.
- [P3] 본문 1문단 마지막 / "the technology we used to track it" / 현재 번역 "이를 추적하는 데 사용한 기술"은 `이를`의 지시 대상이 약간 느슨함 / "기여 내역을 추적하는 데 사용한 기술" 또는 "이 장부를 추적하는 데 사용한 기술"처럼 지시 대상을 명시하는 방향 검토.

## 권장 수정안

- 본문 후반부는 다음처럼 제외 사유를 분리하면 원문의 논리 구조가 더 잘 살아난다: "이 목록에서는 위에서 충분히 부각된 두 주 저자와, 이 책의 첫머리나 표지에 크레딧 순서대로 실려 있는 추천/지지 기여만으로 크레딧을 받은 사람들을 제외했다."
- 첫 문장은 "아래 내용은 프로젝트 내 ⿻ 크레딧의 현재 상태를 보여준다. 이는 본문에서 더 자세히 논의하듯, 우리가 기여를 추적하는 데 사용한 커뮤니티 화폐다."처럼 나누면 스터디 독자가 더 쉽게 읽을 수 있다.

## 보류/주의

- `Credits`를 `크레딧`으로 번역한 것은 장 제목과 본문 용어가 일관되어 있어 유지 가능하다.
- `composite image above`는 이 파일 안에 이미지가 없더라도 원문에 있는 상위 문맥 참조이므로, 현재처럼 "위의 합성 이미지"로 보존하는 것이 타당하다.
- 전체 문체가 평어체 `-다`로 유지되어 다른 번역 장과의 스타일 일관성에는 큰 문제가 없다.
