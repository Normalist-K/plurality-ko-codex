## 권장 수정안

- 인명은 본문 문장 기준으로 `오드리 탕`, `글렌 웨일`, `비탈릭 부테린`처럼 한국어 표기를 우선하고, 링크 텍스트는 원문 유지 여부를 별도 원칙으로 정하면 좋습니다.
- 직함은 스터디 독자를 고려해 `Pehong Chen 컴퓨터과학·통계학 석좌교수`, `맥아더 펠로`, `버그루엔 문화철학상`처럼 핵심 의미를 한국어로 풀되 고유 명칭은 보존하는 방식을 권합니다.
- 예시 수정: `정치적으로 도전적이며` → `정치적 난제가 많고`.
- 예시 수정: `*플루럴리티*를 통해 우리는 이것을...더 깊이...가져가` → `*플루럴리티*를 통해 우리는 이 흐름을...더 깊이 확장해`.
- 예시 수정: `이 모든 것을 더 잘해 낼 길` → `디지털 기술이 자유와 민주주의를 더 제대로 뒷받침하게 할 길`.
- 예시 수정: `Oscars 최초의 케추아 원주민 회원` → `오스카를 주관하는 영화예술과학아카데미의 첫 케추아 원주민 회원`.
