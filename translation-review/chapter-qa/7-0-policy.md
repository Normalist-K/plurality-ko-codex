# 7-0 Policy QA

## 요약 판정

장 전체 번역은 원문의 구조와 핵심 논지를 안정적으로 보존한다. 중대한 누락이나 논지를 뒤집는 오역은 발견하지 못했다. 개선 필요 지점은 주로 정책·법·기술 용어의 정밀도와 영어식 직역 문체에 있다.

- 판정: 조건부 통과
- 이슈 수: P0 0개, P1 0개, P2 9개, P3 6개
- 확인 사항: Markdown heading, 수평선, 번호 목록, 링크 URL, 이미지 경로, figure 구조, 각주 키 보존 확인. code fence는 원문과 번역 모두 없음.

## P0 구조/누락

- 없음. 제목, `---`, 하위 heading, 두 개의 `<figure>`, `<img src>`, `<br></br>`, 번호 목록, 각주 참조/정의가 모두 대응한다.

## P1 의미 오역

- 없음. 장 전체의 정책 모델 비교, ARPA/타이완/DPI 논지, 규제·조세·조직화 논의는 번역에서 대체로 정확히 유지된다.

## P2 용어/문체

- [P2] 7행 / `volunteer investment` / 현재 `자원봉사 투자`는 투자와 자원봉사가 어색하게 결합되어 자발적 기여·투입의 뜻이 흐림 / `자발적 기여`, `자원봉사자들의 투입` 등으로 조정.
- [P2] 17행, 32행 / `technology exports/imports`, `importers of technical frameworks` / 현재 `기술 수출품`, `기술 수입품`, `수입자`는 물품 거래 느낌이 강해 기술 프레임워크의 도입·수용 맥락이 둔해짐 / `수출 기술`, `해외 기술 프레임워크`, `도입자/수용자` 등으로 통일.
- [P2] 28행 / `hard and physical technology` / 현재 `하드 기술과 물리적 기술`은 직역감이 강하고 hard tech의 분야 감각이 약함 / `하드테크 및 물리 기반 기술` 또는 `하드웨어·물리 기술`로 조정.
- [P2] 131행 / `taking a substantial cut of project revenues` / 현재 `프로젝트 수익의 상당 부분을 가져갔음에도`는 착취 뉘앙스가 다소 강함 / `상당한 수수료를 공제했음에도`처럼 중립화.
- [P2] 135행 / `insuperable barriers` / 현재 `장벽을 제공하는 상황`은 한국어 결합이 부자연스러움 / `극복하기 어려운 장벽이 되는 상황`으로 조정.
- [P2] 137행 / `moral and financial rights` / 현재 `도덕적·재정적 권리`는 지식재산권 문맥의 권리 범주를 충분히 살리지 못함 / `인격적·재산적 권리` 또는 저작권 문맥이면 `저작인격권 및 경제적 권리` 검토.
- [P2] 145행 / `environmental counterparty` / 현재 `환경 관련 상대방`은 법·거버넌스 문맥에서 어색함 / `환경 이해관계자` 또는 `환경상 영향을 받는 상대방`으로 조정.
- [P2] 168행 / `data/attention exchange tax` / 현재 `데이터/관심 교환세`는 attention economy의 `attention`이 `관심`으로 읽혀 의미가 흐림 / `데이터/주의력 교환세` 또는 `데이터/주의 교환세`로 조정.
- [P2] 201행 / `enterprises` / 현재 `사업체들`은 기업체로 좁혀 읽힐 수 있음 / `조직적 사업`, `실천체`, `조직들`처럼 영리기업에 한정되지 않게 완화.

## P3 취향/윤문

- [P3] 50행 / `saw their contributions gain national prominence` / 현재 `자신들의 기여가 전국적 주목을 받는 것을 보았다`는 영어식 관찰 구문이 남음 / `기여가 전국적 주목을 받게 되었다`로 자연화.
- [P3] 84행 / `Minister of Digital Transformation`, `Ministry of Digital Affairs`, `Digital Agency` / 현재 링크 텍스트가 영어 직함·기관명 그대로라 학습 흐름이 잠시 끊김 / 원어를 유지하되 한국어 병기를 검토.
- [P3] 101행 / `degree and disciplinary style of rigor and persuasiveness` / 현재 `엄밀성 및 설득력의 수준과 학문 분야 특유의 양식`은 추상어가 겹쳐 읽기 어렵다 / `엄밀성과 설득의 기준이 이상적 사용자 경험과 다르다`처럼 풀기.
- [P3] 115행 / `offerings much more accessible to the world` / 현재 `자신들의 제공물을 세계가 훨씬 더 쉽게 접근할 수 있게`는 어색함 / `전 세계 이용자가 더 쉽게 접근할 수 있게`로 윤문.
- [P3] 189행 / `lifting as many boats as possible` / 현재 `가능한 한 많은 배를 띄워 올리는`은 관용구 직역감이 있음 / `가능한 한 많은 주체가 함께 혜택을 보게 하는` 등으로 의역.
- [P3] 199행 / `For every policy leader... for each one of those` / 현재 `그 한 사람마다`의 선행사가 다소 흐림 / `그 비전을 실제로 구축하는 사람 한 명마다`처럼 지시 대상을 명확화.

## 권장 수정안

- 우선 `attention`, `moral and financial rights`, `environmental counterparty`, `enterprises`처럼 개념 정밀도가 필요한 P2 용어부터 다듬는 것이 좋다.
- `기술 수출품/수입품/수입자` 계열은 장 초반의 디지털 제국 모델 설명에서 반복적으로 인상을 형성하므로 `기술 수출`, `기술 프레임워크 도입/수용` 계열로 정리하는 편이 낫다.
- `2차 투표`, `⿻ 공중`은 다른 한국어 장에서도 쓰이는 용어이므로 이 장에서 독자적으로 바꾸기보다 전체 용어집 결정에 따르는 것이 안전하다.

## 보류/주의

- 원문 Figure 7-0-B의 캡션 시작 태그가 `<figcaptionby>`로 되어 있고 번역도 이를 그대로 보존한다. 번역 누락은 아니지만 렌더링 단계에서 원문 오탈자 추종 여부를 확인할 필요가 있다.
- 링크 URL과 이미지 경로는 대응한다. 일부 bare URL의 마침표 처리 방식은 문장부호 차이로 보이며 링크 누락은 아니다.
- 대상 번역 파일은 수정하지 않았고, QA 리포트 파일만 작성했다.
