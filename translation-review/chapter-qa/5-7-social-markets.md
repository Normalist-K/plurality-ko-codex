## P1 의미 오역

- [P1] 번역 52행, 56행 / 원문 단서: `Payment of marginal returns`, `decreasing marginal returns` / 현재 번역 문제: `한계수익`, `한계수익 체감`은 `marginal revenue`처럼 읽혀, 생산함수의 한계생산/한계수확과 수확체감 조건을 설명하는 원문의 경제학 논지를 흐릴 수 있음 / 권장 수정 방향: 문맥에 따라 `한계생산물 지급`, `한계수확 체감` 또는 `한계생산 체감`으로 조정.

## P2 용어/문체

- [P2] 번역 41행 / 원문 단서: `physical investments` / 현재 번역 문제: `물리적 투자`는 물리학적 투자처럼 어색하게 읽히며 금융시장이 자본을 배분하는 실물 투자 의미가 약함 / 권장 수정 방향: `실물 투자` 또는 `물적 투자`로 수정.
- [P2] 번역 10행, 12행, 85행, 86행 / 원문 단서: `quadratic slider`, `Quadratic Funding`, `quadratic voting` / 현재 번역 문제: 프로젝트 내 용례상 `2차`가 쓰이고 있으나 일반 독자에게는 `secondary`로 오해될 수 있음 / 권장 수정 방향: 용어집에서 유지하기로 한 경우 첫 출현에 `2차(제곱형)` 또는 `2차 펀딩(Quadratic Funding)`처럼 보강하고, 아니라면 `제곱 펀딩/제곱 투표` 계열 검토.
- [P2] 번역 12행 / 원문 단서: `elevates many small contributions across social distance` / 현재 번역 문제: `많은 소액 기여를 높이 평가`는 수식이 소액 기여에 더 큰 가중치를 준다는 제도적 의미보다 태도 평가처럼 읽힘 / 권장 수정 방향: `사회적 거리를 가로지르는 다수의 소액 기여에 더 큰 가중치를 부여하기 때문에`로 수정.
- [P2] 번역 86행 / 원문 단서: `data unions` / 현재 번역 문제: `데이터 조합`은 앞의 협동조합과 의미가 겹치고, 데이터 권리의 집단 교섭 조직이라는 뉘앙스가 약함 / 권장 수정 방향: 프로젝트 용어에 맞춰 `데이터 노조`, `데이터 유니온`, 또는 `데이터 조합(데이터 유니온)`으로 통일.
- [P2] 번역 89행 / 원문 단서: `Economies esteem`, `social esteem/capital` / 현재 번역 문제: `평판 경제`는 reputation economy로 좁혀져, 원문의 인정·존중·사회적 위신의 표지가 화폐처럼 작동한다는 의미가 일부 약해짐 / 권장 수정 방향: `인정 경제`, `사회적 인정 경제`, 또는 첫 문장에서 `평판/인정`을 병기.
- [P2] 번역 108행 / 원문 단서: `property systems that can bring these together` / 현재 번역 문제: 링크 텍스트 `함께 가져올 수 있는`은 직역투이고, 투표 시스템과 재산권 장치를 결합한다는 의미가 흐림 / 권장 수정 방향: `이를 결합할 수 있는` 또는 `이 요소들을 한데 묶는`으로 수정.
- [P2] 번역 111행 / 원문 단서: `exerting authority and predicting what others will decide` / 현재 번역 문제: `권한을 행사하며 다른 이들이 무엇을 결정할지 예측하는 토대를 통해`는 PMP의 권한 행사와 예측 메커니즘이 어떤 역할을 하는지 한 번에 파악하기 어려움 / 권장 수정 방향: `권한 행사와 타인의 결정 예측을 결합한 방식으로`처럼 관계를 명확히 풀어 쓴다.

## 권장 수정안

- `한계수익을 지급하려면` -> `한계생산물에 상응하는 보수를 지급하려면`
- `생산이 "한계수익 체감"을 가져야 한다는 것` -> `생산이 "한계수확 체감"을 보여야 한다는 것`
- `프로젝트와 물리적 투자에 자본을 배분하는` -> `프로젝트와 실물 투자에 자본을 배분하는`
- `2차 펀딩은 사회적 거리를 가로지르는 많은 소액 기여를 높이 평가하기 때문에` -> `2차 펀딩은 사회적 거리를 가로지르는 다수의 소액 기여에 더 큰 가중치를 부여하기 때문에`
- `데이터 조합` -> 용어집 기준에 따라 `데이터 노조` 또는 `데이터 유니온`
- `평판 경제` -> `인정 경제` 또는 `평판/인정 경제`
- `이를 함께 가져올 수 있는 ⿻ 재산 시스템` -> `이 요소들을 결합할 수 있는 ⿻ 재산 시스템`
