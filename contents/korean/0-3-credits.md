# 크레딧

아래는 프로젝트 내 ⿻ 크레딧의 현재 상태를 나타내며, 이는 본문에서 더 자세히 논의하듯 우리가 기여를 추적하는 데 사용한 커뮤니티 통화다. 모든 기여자는 여러 유형의 기여를 했더라도 정확히 하나의 기여 영역으로 분류된다. 범주 내 순서와 전체 글꼴 크기는 2024년 4월 초 기준 ⿻의 정량적 보유량을 나타내며, 위의 합성 이미지를 만들 때도 같은 가중치를 사용했다. 이 목록은 위에서 충분히 강조된 두 주 저자와, 추천사에 대해서만 크레딧을 받은 사람들을 제외한다. 이들은 이 책의 첫머리나 표지에 크레딧 순서대로 나열되어 있기 때문이다. 이를 추적하는 데 사용한 기술의 실험적 성격 때문에 장부에 오류가 있다면 사과드리며, 온라인판과 향후 판본에서 이를 바로잡겠다.
